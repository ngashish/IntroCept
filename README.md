## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Packages

```bash
# Convert objects/arrays into a CSV string or write them into a file. It respects RFC 4180 for the output CSV format.
$ npm i csv-writer

# it's wrapper for csv parser lib what read data from csv files.
$ npm i nest-csv-parser

```
